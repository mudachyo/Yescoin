## How to run  
- Install extension in your browser [Violentmonkey](https://chromewebstore.google.com/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag?hl=be)
- To install **autoclicker** install [this script](https://github.com/mudachyo/Yescoin/raw/main/yescoin-autoclicker.user.js)
- Open [Bot Yescoin](https://web.telegram.org/k/#?tgaddr=tg%3A%2F%2Fresolve%3Fdomain%3DtheYescoin_bot%26appname%3DYescoin%26startapp%3DOx4Ml2) and run the game
